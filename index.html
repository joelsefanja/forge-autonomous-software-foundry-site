The SDLC Agent Pipeline

01
Requirements Agent
Clarifies scope from natural language or problem signal → writes BDD feature files (Gherkin)
02
Architecture Agent
Designs system using Hexagonal Architecture + Laravel + Angular 21. Selects packages. Generates ADRs.
03
Code Generation Agent (OpenCode backbone)
Generates Angular components, Laravel controllers. Enforces code guidelines. Generates OpenAPI spec.
04
Test Generation Agent
Vitest (unit/component/integration) + Specwright → Playwright E2E from feature files + OWASP security scan.
05
UI/UX Research Agent
Heuristic evaluation (Nielsen's 10). Generates solution directions. Outputs PDF report + auto-applies fixes.
06
i18n Agent
Extracts all strings → translation keys. Auto-translates via DeepL / LibreTranslate. Validates RTL support.
07
Compliance Agent
Runs checklist: GDPR/AVG, WCAG 2.2, OWASP, ISO 27001 alignment. Generates compliance PDF.
08
CI/CD Agent
Generates .gitlab-ci.yml + Docker Compose + Coolify config. SSL, domain, monitoring automated.
09
Documentation Agent
README, Swagger API docs, user manual, changelog — human AND AI readable standard.
10
Launch Agent
Landing page + SEO (schema.org, sitemap) + payment integration (Mollie/Stripe/Paddle) + trial logic.
